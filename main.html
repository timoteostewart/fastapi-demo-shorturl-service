This is a URL shortener written in Python using FastAPI with SQLite3 for persistent storage.

Take it for a test drive using its OpenAPI interface.

Alternatively, interact with it via `curl` from a terminal:

PowerSwhell:
curl -X 'POST' `
'https://www.timstewart.io/fastapi-demo-url-shortener/?full_url=https%3A%2F%2Fdev.thnr.net' `
-H 'accept: application/json'

*sh:
curl -X 'POST' \
'https://www.timstewart.io/fastapi-demo-url-shortener/?full_url=https%3A%2F%2Fdev.thnr.net' \
-H 'accept: application/json'
